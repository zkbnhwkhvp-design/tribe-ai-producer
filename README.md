# Tribe GG OS

**Meet Your AI Production Team.**

Tribe GG OS is an AI Creator Operating System for gamers, streamers, and content creators. It combines production planning, creator workflows, hardware management, analytics, and specialized AI agents into one platform.

## Product Pillars

- **AI Workforce:** specialized creator agents for production, audio, video, analytics, thumbnails, sponsors, and community.
- **Creator Dashboard:** one command center for readiness, projects, devices, tasks, and alerts.
- **Studio Hubs:** Audio Hub, Camera Hub, Lighting Hub, Console Hub, Recording Hub, and Stream Hub.
- **Creator Cloud:** accounts, projects, assets, settings, templates, and sync.
- **Marketplace:** future plugins, templates, overlays, presets, and creator tools.

## Starter Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Electron
- Supabase
- PostgreSQL
- Stripe
- Multi-provider AI service layer
