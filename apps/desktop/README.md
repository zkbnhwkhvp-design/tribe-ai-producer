# Tribe GG Desktop App

This app will wrap the web dashboard in Electron and later support native desktop/device capabilities.

## Planned Stack
- Electron
- Shared React UI from apps/web
- Local config storage
- Future native bridge for device discovery and diagnostics

## MVP Desktop Scope
- Launch app shell
- Load dashboard experience
- Persist local settings
- Prepare native bridge abstraction

## Future Scope
- OBS websocket integration
- USB device inspection
- Audio routing helpers
- Smart lighting integrations
- Local recording workflow helpers
