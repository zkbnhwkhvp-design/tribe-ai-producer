# Tribe GG Brand Guide

## Brand Name
Tribe GG

## Product Name
Tribe GG OS

## Tagline
Meet Your AI Production Team.

## Brand Positioning
Tribe GG is an AI-powered creator production platform for gamers and content creators. It should feel like a premium SaaS product, a creator command center, and a gaming-native brand.

## Brand Personality
- Confident
- Technical
- Creator-first
- Premium
- Tactical
- Practical
- Community-driven

## Visual Direction
- Matte white and chrome surfaces
- Black tactical panels
- Frosted glass cards
- Orange energy accents
- Minimal RGB
- Apple-inspired polish
- Gaming UI without looking childish

## Typography

Primary: Inter
Secondary: Space Grotesk
Display Option: Sora

Usage:
- Headlines: Space Grotesk / Sora, bold
- Body: Inter, regular or medium
- UI Labels: Inter, medium
- Metrics: Space Grotesk, semibold

## Logo Usage
- Use full Tribe GG mark for website hero and brand surfaces.
- Use icon-only mark for app sidebar, favicon, and small UI surfaces.
- Maintain clear space equal to the height of the T mark.
- Do not stretch, skew, add random gradients, or use low-contrast placements.

## UI Style
- Rounded cards
- Subtle borders
- Soft elevation
- Glassmorphism used sparingly
- Clear hierarchy
- Dense dashboard layouts with breathing room
- Status colors reserved for meaningful system states

## Photography / Imagery
- Creator setups
- Gaming desks
- Console + capture workflows
- Clean hardware closeups
- Premium dark rooms with controlled light
- Avoid generic esports chaos

## Icon Style
- Rounded line icons
- 1.5px to 2px strokes
- Minimal fills
- Consistent optical size
- Use orange only for active or important states

## Voice
Tribe GG should sound like a sharp production partner: direct, technical, helpful, and ambitious.
