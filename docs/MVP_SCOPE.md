# Tribe GG OS MVP Scope

## MVP Objective
Build a working beta shell that proves the Tribe GG OS product concept: creators can sign in, create projects, view studio readiness, manage mock devices, and interact with an AI Workforce interface.

## MVP In Scope

### Marketing Site
- Home page
- Product overview
- AI Workforce page
- Pricing placeholder
- Download/waitlist CTA
- Contact

### Authentication
- Email/password auth
- User profile
- Workspace creation

### Dashboard
- Creator Readiness score
- Setup status
- Recent projects
- AI Workforce suggestions
- Device health cards
- Quick actions

### Project Management
- Create project
- Project type: stream, video, clip, podcast, campaign
- Project status
- Task checklist

### AI Workforce UI
- Agent roster
- Agent detail pages
- Recommendation cards
- Prompt input panel
- Saved AI recommendations

### Studio Modules
UI-first MVP using mock data:
- Audio Hub
- Camera Hub
- Lighting Hub
- Console Hub
- Device Manager

### Backend
- Supabase schema
- RLS policies
- Seed data
- Basic API contracts

## MVP Out of Scope
- Real-time hardware control
- Native OBS integration
- Full video editing
- Live streaming engine
- Marketplace payments
- Mobile app
- Enterprise permissions

## MVP Demo Script
1. Visit marketing website.
2. Click Get Started.
3. Create account.
4. Complete onboarding.
5. View dashboard.
6. Create a stream project.
7. Open AI Workforce and select Executive Producer.
8. Review setup recommendations.
9. Open Device Manager.
10. Save project.

## Release Criteria
- App runs locally.
- User can authenticate.
- Dashboard loads with realistic data.
- User can create and view projects.
- AI Workforce UI is navigable.
- Core module pages exist.
- Documentation explains setup and product direction.
